### 5.2 解决的问题

| 问题 | 解决方案 |
|------|----------|
| Chaquopy Maven 仓库不可访问 | 使用 Gradle 离线缓存 |
| Kotlin 2.0.20 + AGP 8.7.0 并行编译 state lock | 禁用 `org.gradle.parallel`，添加 `kotlin.compiler.execution.strategy=in-process` |
| compileSdk 36 版本警告 | 添加 `android.suppressUnsupportedCompileSdk=36` |
| 中文路径编译问题 | 添加 `android.overridePathCheck=true` |
| 旧项目代码残留 | 删除 `my_ai_companion/` 等 45 个旧文件 |

### 5.3 构建命令

```bash
cd android
$env:ANDROID_HOME = "F:\Android\SDK"
$env:JAVA_HOME = "F:\Java\jdk-17"
.\gradlew.bat assembleDebug --no-daemon
```
