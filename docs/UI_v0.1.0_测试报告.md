# UI v0.1.0 测试报告

---

## 一、变更概览

| 模块 | 变更内容 |
|------|----------|
| **GlassCard** | 新增自定义View，统一玻璃态卡片，支持深色/亮色主题自动切换 |
| **AppButton.Primary** | 新增粉紫渐变按钮样式，统一主按钮外观 |
| **ParticleView** | 粒子颜色适配深色/亮色模式 |
| **ActivityTransitionHelper** | 新增统一页面转场动画（右进左出） |
| **通用模板** | 新增空状态/加载状态/列表项布局模板 |
| **Drawable 颜色** | 硬编码颜色改为 `@color/` 引用，支持主题切换 |
| **深色模式** | 补充 `values-night/colors.xml` 颜色定义 |

### 涉及页面

- **主页面** (`activity_main.xml`) — 发送按钮改为 MaterialButton
- **设置页面** (`activity_settings.xml`) — 全部卡片改为 GlassCard
- **记忆管理** (`activity_memory_manage.xml`) — 统计栏/搜索栏/列表区改造
- **角色选择** (`activity_character.xml`) — 卡片改造
- **插件管理** (`activity_plugin_manage.xml`) — 已使用 GlassCard

## 二、安装与测试前准备

### 安装 APK

```bash
adb install -r android/app/build/outputs/apk/debug/app-debug.apk
```

或直接将 APK 复制到手机安装。

### 测试环境

- Android 8.0+ 真机或模拟器
- 建议在 **深色模式** 和 **亮色模式** 下各测试一轮
