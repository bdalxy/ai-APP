## 零、前置准备

### 0.1 配置 API Key（必须）

**方式 A：嵌入 APK（推荐，简单）**

1. 复制 `android/app/src/main/python/.env.example` 为 `.env`
2. 用编辑器打开 `.env`，将 `your_api_key_here` 替换为你的 DeepSeek API Key
3. 重新构建 APK

```bash
Copy-Item "android\app\src\main\python\.env.example" "android\app\src\main\python\.env"
notepad "android\app\src\main\python\.env"
# 改 DEEPSEEK_API_KEY=sk-xxxxxxxxx
# 然后重新构建
cd android
.\gradlew.bat assembleDebug --no-daemon
```

## 一、安装 APK

### 1.1 手机准备
1. 设置 → 关于手机 → 连续点击"版本号"7次 → 开启开发者模式
2. 设置 → 开发者选项 → 开启"USB 调试"
3. 数据线连接电脑 → 手机上点"允许 USB 调试"

### 1.2 安装

```bash
# 确认手机已连接
adb devices

# 应显示类似：123456789ABC  device

# 安装 APK
adb install "F:\Trae AI\ai-APP\android\app\build\outputs\apk\debug\app-debug.apk"
```

## 三、P3 聊天功能验证（对话体验）

### 3.1 正常流程

打开应用后，应看到：

1. **标题栏**：显示"小美"（角色名）
2. **状态栏**：显示"就绪"
3. **输入框**：可输入文字
4. **发送按钮**：可点击

### 3.2 发送消息

1. 输入"你好，小美！"
2. 点击发送按钮（→）
3. 状态栏显示"思考中..."
4. 几秒后收到 AI 回复
5. 用户消息蓝色气泡（右对齐），AI 回复灰色气泡（左对齐）

### 3.3 测试建议

| 场景 | 操作 | 预期 |
|------|------|------|
| 基础对话 | "你好，介绍一下你自己" | 小美回复自我介绍 |
| 多轮对话 | 连续发送"今天天气不错"、"你心情好吗" | 上下文连贯，不丢失前文 |
| 新对话 | 点击"新对话"按钮 | 消息列表清空，状态显示"新对话已开始" |
| 空消息 | 输入框为空点发送 | 不发送，无反应 |
| 快速连发 | 快速点击发送多次 | 思考中不会重复发送 |

### 3.4 异常情况

| 场景 | 预期 |
|------|------|
| 未配置 API Key | 状态栏显示"初始化失败: API Key 未配置" |
| 网络断开 | 状态栏显示"错误: ..." 并显示错误消息气泡 |
| 输入过长文字 | 正常情况下应能发送（DeepSeek 有 token 限制） |

## 四、常见问题

### Q: Logcat 不显示日志
A: 输入 `adb logcat -c` 清空日志，再 `adb logcat -s AICompanion:*`

### Q: 应用闪退
A: 查看 Logcat 中 `E` 级别日志，复制错误信息

### Q: 提示"等待中"很久
A: 检查手机网络，确认 DeepSeek API 可访问

### Q: 回复内容不相关
A: 任务角色卡是否正确加载，检查日志中的角色卡信息
