## 一、Chaquopy 插件安装

Chaquopy 是让 Python 代码在 Android 中运行的核心插件。

### 安装步骤

1. 打开 Android Studio
2. `File → Settings → Plugins`
3. 切换到 `Marketplace` 标签
4. 搜索 **"Chaquopy"**
5. 点击 `Install`，安装完成后重启 Android Studio

> 如果 Settings 中没有 Marketplace，先检查网络连接，或到 https://chaquo.com/chaquopy/ 下载离线安装包。

---

## 二、打开 Android Studio

Android Studio 安装在 F 盘，直接双击运行：

```
F:\Android\Android Studio\bin\studio64.exe
```

或者从开始菜单搜索 "Android Studio" 打开。

## 三、打开项目

在 Android Studio 中：
```
File → Open → 选择 F:\Trae AI\开发\android
```

首次打开时，Android Studio 会自动：
- 下载 Gradle 8.9
- 下载项目依赖（Kotlin、AndroidX 等）
- **下载 Chaquopy Python 3.10 运行时**（约 80 MB，首次需 5-10 分钟）

Build → Make Project

## 五、运行验证

将 APK 安装到 Android 设备或模拟器上运行。

### 方式 A：模拟器（推荐）
1. Android Studio → Tools → Device Manager → Create Device
2. 选择 Pixel 7 + API 36 镜像
3. 点击 Run 按钮

### 方式 B：真机
1. 手机开启 开发者选项 → USB 调试
2. USB 连接电脑
3. Android Studio 中点击 Run
