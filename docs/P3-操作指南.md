## 步骤 1：安装 JDK 17

### 1.1 下载

打开浏览器，访问以下链接下载 JDK 17 (Eclipse Temurin)：

**https://adoptium.net/download/**

在页面上选择：
- **Version**: 17 (LTS)
- **Operating System**: Windows
- **Architecture**: x64
- **Package Type**: JDK

点击下载 `.msi` 安装包。

### 1.2 安装

1. 双击下载的 `.msi` 文件
2. 在安装过程中，**关键步骤**：当出现 "Choose Setup Type" 时，选择 **"Custom"** 或 **"Change destination folder"**
3. 将安装路径改为：**`F:\Java\jdk-17`**
4. 继续完成安装（其他选项保持默认即可）

> 如果提示 `F:\Java\jdk-17` 已存在，需要先删除或重命名旧的 JDK 25 文件夹。

---

## 步骤 2：将 "开发" 文件夹改名为 "ai-APP"

> ⚠️ **重要**：文件夹改名前必须关闭所有打开这个文件夹的程序（包括 Trae IDE、Android Studio、文件资源管理器等）。

1. 关闭 **Trae IDE** 和 **Android Studio**（如果开着）
2. 打开文件资源管理器，进入 `F:\Trae AI\`
3. 右键点击 `开发` 文件夹 → 重命名 → 改为 `ai-APP`
4. 确认改名

改名后项目路径变为：
```
F:\Trae AI\ai-APP\android\
```
