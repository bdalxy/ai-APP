## 三、核心架构

```
用户输入 (main.py)
  → RolePlayer.chat()
    → ContextManager（短期记忆，deque）
    → MemoryRetriever（倒排索引预过滤 + 余弦相似度精排）
    → WorldBookEngine（正则匹配注入）
    → PromptBuilder（组装 System Prompt）
    → DeepSeekClient（Chat API，重试+成本追踪）
  → MemoryExtractor（异步提取记忆）
  → 返回 AI 回复

/proactive 命令
  → ProactiveEngine（决策 + 生成）
    → TopicGenerator（话题 Prompt）
    → DeepSeekClient（Chat API）
  → ProactiveScheduler（状态持久化）
```

## 四、操作流程指南

### 1. 配置环境
```powershell
cd "f:\Trae AI\开发\新项目\my_ai_companion"
cp .env.example .env
# 编辑 .env，填入 DEEPSEEK_API_KEY
```

### 2. 启动对话
```powershell
python main.py
# 或指定角色卡
python main.py --card data/role_cards/小美.json
```

### 3. 交互命令
| 命令 | 功能 |
|------|------|
| `/help` | 显示帮助 |
| `/card <路径>` | 切换角色卡 |
| `/world <路径>` | 加载世界书 |
| `/memory` | 记忆统计 |
| `/proactive` | 手动触发主动消息 |
| `/proactive status` | 主动消息状态 |
| `/export` | 导出聊天记录和记忆 |
| `/cost` | 查看 API 费用 |
| `/clear` | 清空对话上下文 |
| `/quit` | 退出 |

### 4. 运行所有测试
```powershell
python -m pytest test/ tests/ -v
```
