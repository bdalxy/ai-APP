## 二、修复详情

### R5: 记忆 Prompt 自然融入引导

**位置**：[prompt_builder.py `_build_memories_section()`](file:///f:/Trae%20AI/ai-APP/src/chat_engine/prompt_builder.py#L200-L218)

**改动前**：
```python
lines = ["## 相关记忆\n"]
for i, memory in enumerate(memories, 1):
    lines.append(f"- {memory}")
```

**改动后**：
```python
lines = [
    "## 关于对方的已知信息\n",
    "以下是你之前了解到的关于对方的信息。请在对话中自然地运用，",
    "不要逐条机械复述，也不要说"根据我的记忆"之类的话——",
    "就像真正的朋友一样，在不经意间自然地提及。\n",
]
```

**效果**：AI 不会生硬地说"根据我的记忆，你喜欢篮球"，而是自然融入。
