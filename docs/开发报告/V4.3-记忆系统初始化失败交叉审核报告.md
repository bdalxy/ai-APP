## 一、根因定位（P0）

### db_path 路径拼接错误

**Kotlin 端** [MainActivity.kt:L402](file:///f:/Trae%20AI/ai-APP/android/app/src/main/java/com/aicompanion/app/MainActivity.kt#L402)：
```kotlin
val dbPath = "${filesDir.absolutePath}/memory.db"     // → /data/data/.../files/memory.db
```

**Python 端** [chat_bridge.py:L247](file:///f:/Trae%20AI/ai-APP/android/app/src/main/python/chat_bridge.py#L247)：
```python
db_file = os.path.join(db_path.rstrip("/").rstrip("\\"), "memories.db")
# → /data/data/.../files/memory.db/memories.db   ❌ 拼接为子路径！
```

`os.path.join` 将 `memory.db` 当作目录，追加 `memories.db` 后路径无效，`sqlite3.connect()` 失败。

### 为何问题隐蔽

```
init_memory() → sqlite3.connect() 失败 → 返回 error JSON
    ↓
extractJsonInt("memory_count") → 字段不存在 → 默认值 -1
    ↓
日志输出 "记忆系统初始化成功，已有 -1 条记忆"  ← 误导！
    ↓
_orchestrator 未设置 → get_memory_stats() 返回 error
    ↓
弹窗显示 "记忆系统未初始化"
```

**层层吞错**：Python try/except → 返回 error JSON → Kotlin 不检查 status → 打印假成功 → 功能静默失效。
