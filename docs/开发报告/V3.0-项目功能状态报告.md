# 项目功能状态报告

**日期**：2026-06-08  
**当前版本**：P3 核心对话完成 + P4 记忆系统编排层就绪

---

## 一、已实现功能

### 1. Android 客户端
| 功能 | 说明 |
|------|------|
| APP 基础框架 | Kotlin + Jetpack Compose，Chaquopy 集成 Python 运行时 |
| 聊天 UI | 对话列表、输入框、发送按钮，支持滚动 |
| 设置界面 | API Key 填写、Token 预设选择（3 种）、模型选择 |
| 配置持久化 | SharedPreferences 存储 API Key / 预设 / 模型 |
| 错误状态处理 | 初始化失败时仍可操作设置，键盘正常调出 |

### 2. 聊天引擎（Python）
| 功能 | 说明 |
|------|------|
| DeepSeek API 集成 | Chat 接口（v4-pro/v4-flash/v4-lite），Embedding 接口 |
| 角色扮演引擎 | role_player.py：加载角色卡、管理上下文、构建 Prompt、调用 API |
| 角色卡系统 | JSON 格式角色卡（"小美"），含人设/性格/风格/称谓等 |
| 上下文管理 | context_manager.py：短期记忆窗口、消息裁剪 |
| Token 预设 | 3 种模式（聊天体验优先/平衡/减少消耗），控制 Prompt 长度、上下文窗口、温度 |
| 用户自选模型 | 可覆盖预设的默认模型，支持 v4-pro/v4-flash/v4-lite |
| API Key 运行时注入 | 从 Android SharedPreferences 传入，不嵌入 APK |

### 3. 记忆系统（已开发，待验证）
| 功能 | 说明 |
|------|------|
| 向量存储 | SQLite 持久化 + 倒排索引 + 余弦相似度，WAL 模式，线程安全 |
| 混合检索 | Embed API + 倒排预过滤 + 余弦精排 + 时间衰减加权 |
| 规则提取 | 正则匹配中文事实和事件（我是/我喜欢/我住在/今天...） |
| LLM 提取 | DeepSeek API 驱动的智能记忆提取（已实现，MVP 未启用） |
| 自动去重 | bigram Jaccard + embedding 余弦相似度 |
| 时间衰减 | 指数衰减（半衰期 7/30/180 天）+ 访问 boost |
| 记忆导出 | JSON 格式导出对话历史/长期记忆/完整数据 |
| 编排层 | MemoryOrchestrator 串联 remember/recall/get_stats 完整链路 |
| 自动存储 | 每次对话后自动提取+存储记忆（chat_bridge 层） |
