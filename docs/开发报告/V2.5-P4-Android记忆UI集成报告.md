## 四、一键对话流程

```
用户输入 → injectMemories(text) → inject_memories → recall → 注入System Prompt
         → chat(text) → 角色扮演 → AI回复
         → _auto_remember → extract + embed + store → 记忆入库
```

---

## 五、错误处理

| 场景 | 行为 |
|------|------|
| 记忆系统初始化失败 | 记录日志，对话框仍可用 |
| 记忆检索失败 | 静默跳过，不阻断对话 |
| 记忆存储失败 | 静默跳过，不阻断对话 |
| 记忆系统未初始化 | 显示提示信息 |
