## 三、最关键的发现

**DeepSeek 没有 `/v1/embeddings` 端点！** embedding API 永远返回 404。

因此做了降级方案：
1. 存储时：embedding 失败 → 跳过向量，继续存储（无向量也能存）
2. 检索时：embedding 失败 → 降级为**倒排索引+关键词匹配**
3. 关键词全不匹配 → 兜底按最近访问时间返回全部记忆

这意味着记忆检索目前是**关键词级别**（非语义级别），但对 P0 验证足够了。
