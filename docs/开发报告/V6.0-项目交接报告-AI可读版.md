## 七、代码风格 & 约定

1. **弹窗**：始终使用 `MaterialAlertDialogBuilder`，不要用原生 `AlertDialog.Builder`
2. **导入**：清理未使用的 import
3. **Message 数据类**：构造参数为 `content`, `isUser`, `isTyping`, `timestamp`（不是 `text`, `sender`）
4. **文件操作**：禁止用 Shell 命令读写文件，必须用 Read/Write/Edit/Grep/Glob 工具
5. **正则**：Kotlin 字符串中 `\s` 必须写成 `\\s`
