## 详细功能说明

### 1. 语音识别（ASR）

- 基于 Android SpeechRecognizer，免费、离线可用
- 支持实时部分结果反馈（输入框实时更新）
- 支持静默超时自动结束（3秒）
- 支持中文/英文/日文语言切换
- 错误处理：无匹配、设备不支持、权限不足

### 2. 语音合成（TTS）

- 基于 Android TextToSpeech，免费、支持中文
- 语速可调：0.5x ~ 2.0x（7档）
- 音调可调：0.5 ~ 2.0（7档）
- 自动朗读：AI回复完成后自动朗读（可在设置中开关）
- 手动朗读：点击消息旁的扬声器图标

### 3. 语音消息

- 基于 MediaRecorder 录制，AAC 格式
- 最长录制时间 60 秒
- 录制时显示音量波形反馈
- 上滑取消发送
- 基于 MediaPlayer 播放，支持进度显示
- 播放/暂停切换

### 4. 语音设置页

- TTS 语速滑块（7档：0.5x ~ 2.0x）
- TTS 音调滑块（7档：0.5 ~ 2.0）
- 自动朗读开关（默认关闭）
- 语音识别语言选择（中文/英文/日文）
- 所有设置持久化到 SharedPreferences

## 已知限制

1. **语音识别依赖系统服务**：部分设备（模拟器）可能不支持语音识别，需真机测试
2. **TTS 引擎依赖**：中文语音合成效果依赖设备安装的中文语音引擎（如 Google TTS 中文包）
3. **语音消息格式**：当前仅支持 AAC 格式，未做格式兼容
4. **自动朗读时机**：仅在 AI 回复完成后触发，不会在流式生成过程中朗读

## 测试指南

### 测试环境
- Android 8.0+ 真机（推荐）
- 已安装 Google TTS 中文语音包
- 已连接网络（首次语音识别可能需联网）

### 测试步骤
1. **语音识别测试**
   - 点击语音按钮（短按 < 300ms）
   - 看到"正在聆听…"提示后说话
   - 识别结果自动填入输入框
   - 检查中文/英文/日文语言切换

2. **语音消息测试**
   - 长按语音按钮（> 300ms）
   - 看到录音提示后说话
   - 松手发送，语音消息气泡出现
   - 点击语音气泡播放，再次点击暂停
   - 长按后上滑取消录音

3. **TTS朗读测试**
   - 发送消息让AI回复
   - 手动点击AI消息旁的扬声器图标
   - 开启自动朗读后发送消息，AI回复自动朗读

4. **语音设置测试**
   - 进入设置 → 语音设置
   - 调整语速滑块，发送消息验证朗读语速
   - 调整音调滑块，验证朗读音调
   - 切换自动朗读开关，验证自动朗读
   - 切换识别语言，验证语音识别语言
   - 退出设置重新进入，验证设置持久化
