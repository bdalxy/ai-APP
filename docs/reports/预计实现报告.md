## 风险提示

| 风险 | 影响 | 缓解措施 |
|------|------|------|
| DeepSeek API 价格调整 | 运营成本增加 | 已有5级 Token 预设，可切换到更经济的模式 |
| Android 系统限制后台任务 | 主动推送不稳定 | 已用 WorkManager + 前台服务双保险 |
| APK 体积增长 | 安装门槛提高 | 避免新增第三方库，使用纯 Python 实现 |
| 多角色独立记忆库 | 存储空间翻倍 | 归档机制自动压缩旧记忆，SQLite 压缩 |
