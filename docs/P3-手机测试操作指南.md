# P3 手机真机测试操作指南

> 更新时间：2026-06-08 | 适用版本：`24f09f7` 及之后

---

## 一、前置条件

| 条件 | 说明 |
|------|------|
| 手机开启 USB 调试 | 设置 → 开发者选项 → USB调试（开） |
| 手机开启 USB 安装 | 小米：开发者选项 → USB安装（开） |
| 数据线连接电脑 | 连上后手机点"允许USB调试" |
| DeepSeek API Key | 没有的话去 https://platform.deepseek.com 注册获取 |

---

## 二、构建 APK（在电脑上执行）

### 步骤 1：打开 PowerShell

按 `Win+R`，输入 `powershell`，回车。

### 步骤 2：进入项目目录

```powershell
cd "F:\Trae AI\ai-APP\android"
```

### 步骤 3：设置环境变量并构建

复制下面整段粘贴，回车：

```powershell
$env:ANDROID_HOME = "F:\Android\SDK"
$env:JAVA_HOME = "F:\Java\jdk-17"
$env:Path = "$env:JAVA_HOME\bin;$env:Path"
.\gradlew.bat assembleDebug --no-daemon
```

等一下，看到 `BUILD SUCCESSFUL` 就行了（约 2-5 分钟）。

---

## 三、安装到手机（在电脑上执行）

### 步骤 4：确认手机已连接

```powershell
& "F:\Android\SDK\platform-tools\adb.exe" devices
```

应该看到类似：

```
List of devices attached
1dafb92f        device
```

如果 `1dafb92f` 换成别的设备ID，说明连接成功。

### 步骤 5：安装 APK

```powershell
& "F:\Android\SDK\platform-tools\adb.exe" install -r -d "F:\Trae AI\ai-APP\android\app\build\outputs\apk\debug\app-debug.apk"
```

看到 `Success` 就安装好了。

---

## 四、测试步骤（在手机上操作）

### 步骤 6：打开 APP

在手机桌面找到"AI Companion"图标，点击打开。

### 步骤 7：输入 API Key

首次打开会弹出"配置 API Key"对话框：
- 输入你的 DeepSeek API Key（`sk-` 开头的一串）
- 点"确认"

### 步骤 8：等待初始化

看到左下角状态从"正在初始化..."变成"就绪"就行了。

### 步骤 9：发送消息

在底部输入框输入一句话，如"你好，小美"，点发送按钮（→）。

---

## 五、合格标准

| 测试项 | 预期结果 |
|--------|----------|
| 打开 APP | 弹出 API Key 输入框 |
| 输入 API Key 后点确认 | 状态显示"正在初始化..." → "就绪" |
| 发送"你好" | AI 回复角色对话（不是"作为AI..."） |
| 发送"你是谁" | AI 回复角色设定内容（如"我叫小美"） |
| 点"新对话" | 消息列表清空，状态显示"新对话已开始" |
| 点齿轮图标⚙ | 弹出设置菜单（预设/模型/API Key） |
| 切换Token预设 | 状态提示"已切换为..." |
| 切换模型 | 状态提示"模型已切换为..." |

## 六、查看日志（出问题时用）

在电脑上运行：

```powershell
& "F:\Android\SDK\platform-tools\adb.exe" logcat -s AICompanion:* System.out:*
```

- 正常：看到 `Python 初始化结果` 和 `AI 回复`
- 异常：看到 `Python 初始化失败` 或 `RuntimeException`，把错误信息发给我

---

## 七、常见问题

| 问题 | 解决方法 |
|------|----------|
| `adb: command not found` | 用完整路径 `F:\Android\SDK\platform-tools\adb.exe` |
| `INSTALL_FAILED_USER_RESTRICTED` | 手机开发者选项里开"USB安装" |
| 初始化失败 | 检查 API Key 是否正确，网络是否正常 |
| 回复内容不对 | 检查 API Key 余额是否充足 |
| 闪退 | 运行日志命令，把错误信息发给我 |
