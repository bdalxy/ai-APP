## 二、数据流设计

```
用户输入 → Kotlin(MainActivity.sendMessageStream)
  → Chaquopy → chat_bridge.chat_stream_start()
    → 插件预处理 → _inject_context(记忆+世界书)
    → RolePlayer.chat_stream() → DeepSeekClient.chat_stream()
      → DeepSeek API (SSE)
    → token_queue.put(token)
  → Kotlin 轮询: chat_stream_poll(stream_id)
    → batch 返回 → UI 逐 token 更新
    → done → 保存对话 + 异步记忆存储
```
