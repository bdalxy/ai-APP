## 三、主题样式

### [themes.xml](file:///f:/Trae%20AI/ai-APP/android/app/src/main/res/values/themes.xml)

**核心主题**：`Theme.AICompanion`

```xml
<style name="Theme.AICompanion" parent="Theme.MaterialComponents.NoActionBar">
    <item name="colorPrimary">@color/primary</item>
    <item name="android:statusBarColor">@color/bg_dark</item>
    <item name="android:windowBackground">@color/bg_dark</item>
    <item name="android:windowLightStatusBar">true</item>
</style>
```

**关键配置**：
- 无 ActionBar（`NoActionBar`），自定义顶栏
- 深色状态栏背景
- 亮色状态栏图标（`windowLightStatusBar=true`）

## 九、关键设计原则

1. **玻璃态**：半透明背景 + 1dp 白色边框 + 12dp 圆角，营造"浮在空间"的感觉
2. **紫色系**：主色 #9B59B6，辅助色 #E8D5F5，深色背景 #1A0A2E
3. **Material Design 3**：基于 MD3 组件体系，使用 NoActionBar 主题
4. **响应式**：通过 sw600dp/sw840dp 适配平板，横屏有独立布局
5. **矢量图标**：所有图标使用 Vector Drawable，无损缩放
