## 三、主题样式

### [themes.xml](file:///f:/Trae%20AI/ai-APP/android/app/src/main/res/values/themes.xml)

**核心主题**：`Theme.AICompanion`

```xml
<style name="Theme.AICompanion" parent="Theme.MaterialComponents.NoActionBar">
    <item name="colorPrimary">@color/primary</item>          <!-- 淡天蓝 #B0C4DE -->
    <item name="android:statusBarColor">@color/bg_dark</item> <!-- 淡樱粉 -->
    <item name="android:windowBackground">@color/bg_dark</item>
    <item name="android:windowLightStatusBar">true</item>     <!-- 亮色状态栏图标 -->
</style>
```

## 九、关键设计原则

1. **玻璃态**：半透明背景 + 1dp 淡天蓝边框 + 12dp 圆角，营造"浮在空间"的感觉
2. **樱羽四色**：淡樱粉(#FDF0F0) 40% + 淡粉蓝(#D4E8F0) 25% + 淡天蓝(#B0C4DE) 20% + 天使白(#FFFFFF) 15%
3. **Material Design 3**：基于 MD3 组件体系，使用 NoActionBar 主题
4. **响应式**：通过 sw600dp/sw840dp 适配平板，横屏有独立尺寸
5. **矢量图标**：所有图标使用 Vector Drawable，无损缩放
6. **onlyDark 消色差**：主题色使用 `theme` 属性引用，自动适配深浅模式
