## 快速入门（30分钟）

### 第一步：了解项目是什么（5分钟）

**项目名称**：AI Companion（AI 伙伴）  
**核心功能**：AI 角色扮演聊天 APP，支持长期记忆、世界书、插件系统、语音对话、主动消息推送  
**技术栈**：Kotlin 2.0.20 + Python 3.10（Chaquopy 桥接）+ DeepSeek API  
**记忆系统**：v2.0（11 细粒度分类，从 13 类合并优化）  
**世界书**：支持优先级冲突处理（入口优先级 + 全局赋值优先级）  
**设计主题**：樱羽主题 v1.0 — 淡樱粉、淡粉蓝、淡天蓝晕染天使白，半透明玻璃态  
**主色调**：淡天蓝 `#B0C4DE` (primary)，淡樱粉 `#FFDCE3` (secondary)

> 阅读 [01_项目架构概览](learning-reports/01_项目架构概览.md) 了解完整架构

### 第二步：了解数据怎么流动（10分钟）

```
用户输入 → MainActivity.kt → ChatViewModel.kt
    → Python chat_bridge._core.chat_stream_start()
        → RolePlayer.chat()
            → 检索记忆 → 匹配世界书 → 构建Prompt → DeepSeek API
        → 流式返回 token 队列
    → ChatViewModel 轮询取 token
    → ChatAdapter 逐字更新 UI
    → 对话结束后自动提取记忆 → 存入 SQLite 向量库
```
