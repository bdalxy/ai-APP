## P0 修复详情

### P0-01: init() 参数错位

**文件**: `MainActivity.kt:587-599`

**问题**: Kotlin 端只传 5 个参数给 Python 端 8 参数的 `init()` 函数，导致 `max_tokens`、`model`、`example_dialogues` 用户配置失效，`top_p` 被设为 1000（应为 0.9）。

**修复**: 补齐 `topP`、`freqPenalty`、`presPenalty` 三个参数。

```kotlin
// 修复前
module.callAttr("init", ctxSize, temp, maxTk, dialogues, model)

// 修复后
module.callAttr("init", ctxSize, temp, topP, freqPenalty, presPenalty, maxTk, dialogues, model)
```

### P0-02: ChatViewModel 重构为 AndroidViewModel

**文件**: `ChatViewModel.kt` + `MainActivity.kt`

**问题**: ChatViewModel 不继承 ViewModel，持有 Context/Binding 引用，违反 MVVM 原则。

**修复**:
- 继承 `AndroidViewModel`，移除 `context` 和 `binding` 参数
- 新增 15 个 LiveData 字段暴露 UI 状态
- 使用 `viewModelScope` 替代 `lifecycleScope`
- MainActivity 通过 ViewModelProvider 创建并观察 LiveData

### P0-03: WorldBookSection.kt 硬编码字符串迁移

**文件**: `WorldBookSection.kt` + `strings.xml`

**问题**: ~713 行硬编码中文字符串。

**修复**: 新增 72 个 `world_book_` 前缀字符串资源，全部替换为 `activity.getString(R.string.world_book_xxx)`。

### P0-04: SettingsDetailActivity.kt 硬编码字符串迁移

**文件**: `SettingsDetailActivity.kt` + `strings.xml`

**问题**: ~150 行硬编码中文字符串。

**修复**: 新增 50+ 个字符串资源，全部替换为 `getString(R.string.xxx)`。

## P1 修复详情

### P1-01: 统一插件体系

**文件**: `plugin/IPlugin.kt`, `plugin/PluginRegistry.kt`, `plugin/BuiltinPlugins.kt`, `plugin/PluginInfo.kt`

**问题**: Kotlin 端插件体系（空壳桩）与 Python 端插件体系双重存在。

**修复**: 全部标记 `@Deprecated("迁移到 Python 端插件体系")`，保留向后兼容。

### P1-02: 修复 inner class 内存泄漏

**文件**: `ChatViewModel.kt`（通过 P0-02 重构解决）

**问题**: SearchHelper、StreamingHelper 作为 inner class 持有外部 Activity 引用。

**修复**: ViewModel 重构后，inner class 持有 ViewModel 引用（生命周期由框架管理），不再有泄漏风险。

### P1-03: 语音录音加密

**文件**: `VoiceRecorder.kt`, `VoicePlayer.kt`, `VoiceController.kt`, `AICompanionApp.kt`

**问题**: 语音录音文件明文存储，未加密。

**修复**:
- 使用 AES-256-GCM（AndroidX EncryptedFile + Android Keystore）加密存储
- 录音完成后自动加密并删除明文原始文件
- 播放时解密到临时文件，播放完成后立即删除

### P1-04: 录音文件自动清理

**文件**: `VoiceRecorder.kt`, `VoiceController.kt`, `AICompanionApp.kt`

**问题**: 录音文件使用后未自动清理。

**修复**:
- `cleanupOldRecordings()` 清理超过 24 小时的过期录音
- 触发点：启动时、录制完成后、销毁时

### P1-05: WorldBookEntry 双重定义

**文件**: `WorldBookEntry.kt`, `module/worldbook/WorldBookModule.kt`, `WorldBookActivity.kt`

**问题**: `app.WorldBookEntry` 和 `module.worldbook.WorldBookEntry` 两套不同定义共存。

**修复**: 统一为 `app.WorldBookEntry`，module 中使用 `typealias` 引用。

### P1-06: CharacterData 补充 Python 端字段

**文件**: `CharacterData.kt`, `CharacterStorage.kt`

**问题**: CharacterData 缺少角色卡 JSON 中的 7 个字段。

**修复**: 新增 `nickname`、`age`、`gender`、`likes`、`dislikes`、`exampleDialogues`、`creatorNotes` 字段。

### P1-07: PluginInfo 数据完整性

**文件**: `plugin/PluginInfo.kt`, `PluginManageActivity.kt`

**问题**: PluginInfo 丢弃了 Python 端插件的大量元数据。

**修复**: 新增 `category`、`dependencies`、`conflicts`、`callCount`、`errorCount`、`installTime`、`lastCallTime`、`lastError` 共 8 个字段。
