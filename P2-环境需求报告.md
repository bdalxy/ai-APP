## 二、需安装的工具（按顺序）

### 1. JDK 17（Java Development Kit）

| 项目 | 说明 |
|------|------|
| **作用** | Kotlin 编译运行的基础，Android 开发必需 |
| **版本要求** | JDK 17（LTS 长期支持版） |
| **下载地址** | https://adoptium.net/download/ (推荐 Eclipse Temurin) |
| **预计大小** | ~200 MB |
| **安装步骤** | 1. 下载 JDK 17 MSI 安装包<br>2. 以管理员身份运行安装<br>3. 安装后重启终端验证 |
| **验证命令** | `java -version` → 应显示 `17.x.x` |
| **备注** | Android Studio 自带的 JDK 也可以，但单独安装更稳定 |

### 2. Android Studio

| 项目 | 说明 |
|------|------|
| **作用** | Android 官方 IDE，提供项目模板、编译、模拟器、调试、APK 打包 |
| **版本要求** | Hedgehog (2023.1.1) 或更新 |
| **下载地址** | https://developer.android.com/studio |
| **预计大小** | ~1.5 GB（安装后 ~4 GB） |
| **安装步骤** | 1. 下载 Android Studio 安装程序<br>2. 运行安装，选择 Standard 安装模式<br>3. 安装过程会自动下载 Android SDK 34+<br>4. 首次启动会自动配置 |
| **验证方式** | 能创建并运行 Empty Activity 项目 |

### 3. Android SDK 34+

| 项目 | 说明 |
|------|------|
| **作用** | 编译 Android APK 所需的库和工具链 |
| **版本要求** | API Level 34（Android 14） |
| **安装方式** | Android Studio 安装时自动下载，或通过 SDK Manager 手动安装 |
| **预计大小** | ~2 GB |
| **验证方式** | Android Studio → SDK Manager → 确认 API 34 已勾选 |

### 4. Chaquopy 插件

| 项目 | 说明 |
|------|------|
| **作用** | 将 Python 嵌入 Android 应用的核心插件，让 Kotlin 能直接调用 Python 代码 |
| **版本要求** | 最新稳定版（15.x+） |
| **安装方式** | Android Studio → Settings → Plugins → 搜索 "Chaquopy" → 安装 |
| **预计大小** | ~10 MB（插件本体） |
| **验证方式** | 创建项目后，在 `build.gradle` 中能正确引用 Chaquopy 依赖 |
| **备注** | 这是项目技术方案的核心 — 让 Python 的 AI 逻辑在 Android 上运行 |

## 五、P2 阶段验证内容

安装完成后，P2 阶段需要验证：

1. **Python 代码在 Android 中运行** — 创建 Android 项目，通过 Chaquopy 调用 Python 模块
